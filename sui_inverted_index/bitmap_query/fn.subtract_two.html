Merge-join subtraction for an anchored negative literal.

A negative literal in a valid term is evaluated as a AND NOT b, where a is the already-anchored positive candidate stream. Bitmap subtraction (a_bm - b_bm) has the same truth table without materializing NOT b over the whole range:

a b | a AND NOT b
1 0 | 1
0 1 | 0
0 0 | 0
1 1 | 0

For each bucket in a, emits a_bm - b_bm if b has the same bucket, else emits a_bm unchanged. Drops empty results.
